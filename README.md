# How to start with docker

Up: 
```bash
docker-compose up -d 
```

Go: http://localhost/

# How to start without docker


## Frontend
    
```bash
# 1. install dependencies
npm install
# 2. run server
npm run start
```

## Backend

```bash
# 1. go to backend folder
cd backend
# 2. install dependencies
npm install
# 3. run server
node server.js
```
