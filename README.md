# How to start with docker

Up: 
```bash
docker-compose up -d 
```

Go: http://localhost/

# How to start without docker

## 1. Install dependencies

```bash
npm install
```

## 2. Start frontend
    
```bash
npm run start
```

## 3. Start backend

```bash
node backend/server.js
```
